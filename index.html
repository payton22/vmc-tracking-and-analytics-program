The Veteran Military Center Tracking and Analytics Program (VMC-TAP) is a software data gathering and analytics program. The goal of this program is to collect, store, and combine data from several sources into a single usable database. The program will automate tracking of the rate and duration of students at events as well as the Veteran's Military Centers' (VMC) locations. The program will also combine all the data collected from various sources that can be queried for data visualization. The data capture and visualization are important to the center's existence and helps determining the success of events as well as requests for funding.
